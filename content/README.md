# Content folder (admin-editable)

Edit these JSON files to update the website. See **docs/ADMIN.md** for full instructions.

| File | What it controls |
|------|------------------|
| `site.json` | Phone, email, address, hours, social links |
| `properties.json` | All sale & rent listings |
| `testimonials.json` | Homepage reviews |
| `team.json` | About page staff |
| `areas.json` | Catford, Lee, Hither Green, Lewisham guides |
| `why-choose.json` | Homepage “Why Choose Kingswell” |
| `blog.json` | Blog / property insights articles |

After editing: commit & push to Git → Vercel redeploys automatically.
