# Images folder

Upload images here, then reference them in `content/*.json` with a leading slash:

- `/images/properties/my-listing/front.jpg`
- `/images/team/marcus.jpg`
- `/images/blog/market-update.jpg`

Subfolders:
- `properties/` — listing photos
- `team/` — staff headshots
- `blog/` — blog hero images

Keep filenames lowercase with hyphens (e.g. `rushey-green-house.jpg`).
